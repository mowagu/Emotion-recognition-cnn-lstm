# Emotion Recognition using CNN + LSTM

This project detects human emotions from speech audio using deep learning.

---

# Model Architecture
- CNN (Conv1D layers)
- LSTM layers
- Dense layers

---

# Dataset
- CREMA-D dataset

---

#  Accuracy
- Test Accuracy: ~90%
- Train Accuracy: ~99%

---

# Features Used
- MFCC
- Delta
- Delta-Delta
- Chroma
